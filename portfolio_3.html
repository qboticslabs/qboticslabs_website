Autonomous Robot Butler

Home Portfolio Autonomous robot butler

Description

This project was a internal research project which was mainly done for our book "Learning Robotics using Python". The aim of the project was to create a robot butler which can carry food and beverages and serve to people inside the hotel. The complete software of the robot is done on R.O.S and it perform autonomous navigation using algorithms such as SLAM and AMCL.
